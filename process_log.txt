2. Flask Backend Development
================================================================================

Implemented a Flask application to serve the frontend and provide API endpoints.

Added:
- "/" route → renders index.html
- "/api/search" → searches movies using TMDB API
- "/api/movie/<id>" → returns detailed movie information

Used the requests library to call TMDB APIs.

================================================================================
3. API Key Management
================================================================================

Stored TMDB API key using environment variables instead of hardcoding.

Used:
export TMDB_API_KEY=...

Accessed in Flask using:
os.environ.get("TMDB_API_KEY")

Ensured the key is not included in GitHub repository.
